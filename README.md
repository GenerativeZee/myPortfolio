## ✨ Features
- **Dark Futuristic Aesthetic:** High-end startup UI inspired by OpenAI, Linear, and Vercel.
- **Cinematic Animations:** Powered by Framer Motion, featuring smooth stagger transitions, ambient dynamic glows, and scroll-triggered reveals.
- **Interactive Command Palette:** Press `⌘K` or `Ctrl+K` for fast cross-page navigation.
- **Dynamic Projects & Experience:** Advanced interactive timeline grids and project cards showcasing hybrid RAG systems, LLM orchestration, and NLP pipelines.
- **High-Performance:** Fully optimized Next.js App Router architecture with custom-built lightweight utility components.
- **SEO Optimized:** Structured semantic HTML5 with built-in OpenGraph and metadata configuration.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (with custom global design system tokens)
- **Motion:** Framer Motion
- **Icons:** Lucide React & Custom SVGs
- **Deployment:** Ready for Vercel

## ⚙️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GenerativeZee/myPortfolio.git
   cd myPortfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure
- `src/app/` - Next.js routing, layouts, and global CSS definitions.
- `src/components/` - Reusable UI components (Navbar, LoadingScreen, Animations, Icons).
- `src/components/sections/` - Distinct portfolio sections (Hero, Experience, Projects, Skills, Contact).

## 📝 Customization
- **Theme Variables:** Modify `src/app/globals.css` to update the dark theme colors and glows.
- **Content:** Update the text/data constants in the respective `src/components/sections/*.tsx` files to personalize your resume data.
